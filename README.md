## Prerequisites

## Installation & Setup

1. Clone the repo:

   git clone https://github.com/Geethuharshan7/country-list.git

2. Install dependencies:
   npm install

3. Run the project:
	npm start
    The app will run at http://localhost:3000

4. Production build:
    npm run build
